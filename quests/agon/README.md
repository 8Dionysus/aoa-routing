# Agon Quest Lane

This lane holds active Agon gate-routing follow-through records for
`aoa-routing` when they use current repo-qualified IDs.

These records can point to routing-owned Agon parts, but they do not grant
arena, verdict, scar, rank, summon, retention, ToS, or runtime authority.
`Agents-of-Abyss` remains the stronger Agon owner.

Historical Agon receipts are not current quest-lane entries. They are reached
through `../../mechanics/agon/PROVENANCE.md`.
