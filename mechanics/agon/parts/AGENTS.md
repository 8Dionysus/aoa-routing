# Agon Parts Guidance

Each directory here is an active operation part. Do not create file-type
buckets above the part. If a file belongs to an Agon route operation, place it
inside the owning part's `docs/`, `config/`, `schemas/`, `examples/`,
`generated/`, `scripts/`, `tests/`, or `manifests/` subtree.

Source quest records stay in root `quests/`, not in part-local `quests/`
directories.
