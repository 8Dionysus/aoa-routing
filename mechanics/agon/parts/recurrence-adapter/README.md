# Agon Recurrence Adapter

This part owns observation-only recurrence exposure for Agon gate-routing
surfaces.

| Field | Value |
| --- | --- |
| Role | Let recurrence notice repeated Agon gate pressure without activating an arena. |
| Input | Gate-routing docs and generated registry. |
| Output | Recurrence component manifest and hook binding set. |
| Owner | `aoa-routing` for local observation surfaces; recurrence remains a separate control plane. |
| Next route | Refresh gate-routing validation or hand off review pressure to the center. |
| Tools | Manifest review plus Agon gate-routing validator. |
| Verification | Validate that manifest paths point to active Agon part surfaces. |

The adapter may produce observation pressure only. It must not create arena
sessions, verdicts, scars, retention schedules, rank mutations, Tree-of-Sophia
promotion, or automatic source rewrites.
