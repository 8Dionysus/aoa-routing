# Agon Legacy

This directory preserves former Agon root-path accounting. It is not an active
route and must not become a compatibility fallback.

## Layout

- `INDEX.md`: former root paths mapped to active part routes.
- `DISTILLATION_LOG.md`: dated accounting for this refactor.
- `raw/`: historical receipt material, including the former Wave V landing
  note.

Routine work starts in `../parts/`. Enter legacy through `../PROVENANCE.md`
only when old-path history matters.
