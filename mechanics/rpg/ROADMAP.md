# RPG Roadmap

## Current Contour

RPG is present as an example-only navigation bridge.

The mechanic can project quest, party, and unlock surfaces as inspect/expand/
handoff hints. It does not own reward logic, progression state, party truth, or
quest meaning.

## Next Work

- Keep `navigation-bridge` example-only and validator-shaped.
- Keep source inputs pointed at playbooks, agents, and evals rather than
  routing-owned quest dispatch.
- Route repeated RPG pressure to owner repos before adding live builders.

## When Time Comes

- Add live builders only after source owners publish real generated surfaces.
- Add stronger RPG route readers when repeated navigation use needs public
  routing output.

## Out Of Scope

- Reward verbs.
- Completion writes.
- Party state authority.
- Quest source authority.
