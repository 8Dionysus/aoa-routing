# Titan Roadmap

## Current Contour

Titan is present as runtime route, console route, memory route, appserver
bridge, and runtime case routing.

The mechanic names Titan next routes and gates. It does not become Titan
identity, runtime, memory, proof, SDK bridge, or appserver authority.

## Next Work

- Keep Titan route schemas aligned with part-local examples and tests.
- Keep runtime case routing weaker than runtime owner evidence.
- Keep memory route hints weaker than memo source truth.

## When Time Comes

- Add owner-local Titan receipts when stronger owners accept route contracts.
- Add generated Titan readers only if multiple Titan parts need one public
  routing surface.

## Out Of Scope

- Summoning or mutating Titan.
- Runtime execution.
- Memory writes.
- Proof or stats authority.
